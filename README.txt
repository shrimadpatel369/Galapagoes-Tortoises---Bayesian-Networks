To run the project follow the steps below

1. Unzip (Extract Files)
2. Go into the folder :	'Project_Final_Submission'
3. Open the Survival_Predictor.ipynb
4. Ensure all the required libraries are installed beforehand. Required Libraries are provided in the requirement.txt file. 
	Run the requirement.txt file using the command on the command prompt: "pip install -r requirement.txt"
5. Click Restart and Run All.

6. Done
